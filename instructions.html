How to install GhostPop in Chrome

This guide is for people who have never installed a Chrome extension from a ZIP file before.

Before you start

  • Use Google Chrome on your computer.
  • Download the GhostPop ZIP file from this site first.
  • Do not unzip it and then re-zip it later. Just extract it once when Chrome needs the folder.
1

Download the extension ZIP

Click Download ZIP. Your browser should save a file named depop-renewal-extension.zip or ghostpop-extension.zip.

2

Extract the ZIP file

Find the downloaded ZIP on your computer, right-click it, and choose Extract or Unzip. This should create a regular folder with the extension files inside.

3

Open Chrome Extensions

In Chrome, type chrome://extensions into the address bar and press Enter.

4

Turn on Developer mode

On the Extensions page, find the Developer mode switch in the top-right corner and turn it on. This is required for any extension you install from your own computer.

5

Load the extracted folder

Click Load unpacked, then select the folder you extracted from the ZIP file. Choose the folder itself, not a file inside it.

6

Pin GhostPop to your toolbar

Click the puzzle-piece icon in Chrome, find GhostPop, and pin it so it stays easy to find.

7

Log into Depop in Chrome

Open Depop in Chrome and sign into the account you want to use. GhostPop needs your active login session to renew listings.

8

Open GhostPop and follow the prompts

Click the GhostPop icon in your Chrome toolbar to open it. From there you can scrape listings, choose a schedule, or run a renewal manually.

If Chrome says the extension is invalid

Make sure you selected the extracted folder, not the ZIP file. Chrome cannot install the ZIP directly with the Load unpacked button.

If you cannot find the extension icon

Click the puzzle-piece icon near the top-right of Chrome, then pin GhostPop so it always stays visible.

If renewals do not work

Check that you are still logged into Depop in Chrome. If you were signed out, GhostPop will not be able to renew your listings.

Need the file again?

Download the extension and come back to this guide anytime.
